🚀 DEPLOYMENT GUIDE

Launch SprintVibe

Your complete step-by-step guide to publishing your app live — no engineering experience needed.

✅ PC / Mac
✅ iPhone (iOS)
✅ Android
✅ Tablet
⏱

Total time: ~15 minutes

From zero to a live URL your team can open on any device

PHASE 1 — INSTALL THE FREE TOOLS (5 min)
1

Install Node.js

Node.js is the engine that builds and runs your app. It's free and takes 2 minutes to install.
💡
Go to nodejs.org → click the big green "LTS" button → download → open the installer → click Next all the way through. That's it.
🔍
To check it worked: open Terminal (Mac) or Command Prompt (Windows) and type node --version — you should see a number like v20.x.x
2

Create a free GitHub account

GitHub stores your code and connects to Vercel for automatic deployments.
💡
Go to github.com → Sign up → choose the free plan. Remember your username — you'll need it.
3

Create a free Vercel account

Vercel is what publishes your app to a live URL in seconds. 100% free for personal projects.
💡
Go to vercel.com → click "Sign Up" → choose "Continue with GitHub" → this links them together automatically.
PHASE 2 — SET UP YOUR PROJECT (5 min)
4

Download the project files

I've prepared the complete project folder. Download the ZIP file attached to this guide — it contains everything pre-configured.
📁 sprintvibe/
📁 src/
✨ App.jsx  ← your entire app
✨ main.jsx  ← app entry point
📁 public/
✨ favicon.svg  ← app icon
✨ index.html  ← HTML shell
✨ package.json  ← dependencies list
✨ vite.config.js  ← build config + PWA
✨ vercel.json  ← deploy config
✨ .gitignore  ← files to ignore
📦
Unzip the folder somewhere easy to find — like your Desktop.
5

Open Terminal in the project folder

The Terminal is how you run commands. Don't worry — you only need to type 3 short things total.
🍎
Mac: Right-click the sprintvibe folder → "New Terminal at Folder"
(If you don't see that: open Terminal app, type cd then drag the folder into Terminal and press Enter)
🪟
Windows: Open the sprintvibe folder → click the address bar at the top → type cmd → press Enter
6

Install the app's dependencies

This downloads the code libraries your app needs. Run this once — it takes about 30 seconds.
PASTE THIS INTO TERMINAL AND PRESS ENTER
npm install
✅
You'll see a node_modules folder appear. That means it worked.
7

Test it locally on your computer first

Before publishing, preview the app on your own machine to make sure everything looks right.
RUN THIS COMMAND
npm run dev
✅
Terminal will show a URL like http://localhost:5173 — open that in your browser and you'll see SprintVibe running!
💡
To stop the local server, press Ctrl + C in Terminal.
PHASE 3 — PUBLISH TO THE INTERNET (5 min)
8

Create a GitHub repository

A repository is just a folder on GitHub that stores your code.
💡
1. Go to github.com → click the + button (top right) → New repository
2. Name it sprintvibe
3. Make sure it's set to Public
4. Click Create repository
5. GitHub will show you a page with commands — copy the URL shown (looks like https://github.com/YOUR-NAME/sprintvibe.git)
9

Push your code to GitHub

This uploads all your files to GitHub. Run these 4 commands in order — replace the URL with yours from Step 8.
RUN THESE 4 COMMANDS ONE AT A TIME
git init
git add .
git commit -m "Launch SprintVibe"
git remote add origin https://github.com/YOUR-NAME/sprintvibe.git
git push -u origin main
⚠️
Replace YOUR-NAME in the 4th command with your actual GitHub username.
✅
Refresh your GitHub page — you should see all your files uploaded there.
10

Deploy to Vercel — get your live URL

This is the final step. Vercel reads your GitHub code and publishes it live in about 60 seconds.
💡
1. Go to vercel.com → click Add New → Project
2. Find sprintvibe in the list → click Import
3. Leave all settings as default → click Deploy
4. Wait ~60 seconds for the confetti screen 🎉
🎉
You're live! Vercel gives you a URL like:
https://sprintvibe-yourname.vercel.app

Share this with anyone — it works on every device instantly.
PHASE 4 — INSTALL ON PHONES (like a real app)
📱

Add to Home Screen on iPhone

This makes SprintVibe feel like a native app — full screen, no browser bar, with your icon.
🍎
1. Open Safari on iPhone (must be Safari, not Chrome)
2. Go to your Vercel URL
3. Tap the Share button (the box with an arrow pointing up)
4. Scroll down and tap "Add to Home Screen"
5. Tap Add — SprintVibe icon appears on your home screen!
🤖

Install on Android

Android makes this even easier — Chrome detects it automatically.
🤖
1. Open Chrome on Android
2. Go to your Vercel URL
3. Tap the ⋮ menu (three dots, top right)
4. Tap "Install App" or "Add to Home Screen"
5. Tap Install — done!
MAKING CHANGES IN THE FUTURE
🔄

How to update your app

Any time you make changes to the code, just run these 3 commands and Vercel auto-deploys in 60 seconds.
DEPLOY AN UPDATE
git add .
git commit -m "describe what you changed"
git push
✅
Vercel detects the push automatically and re-deploys. Your live URL updates within a minute — no other steps needed.
IF SOMETHING GOES WRONG
🛠

Common fixes

Most issues have simple solutions.
❓
"npm: command not found" — Node.js didn't install correctly. Restart your Terminal and try again, or re-download Node from nodejs.org.
❓
"Permission denied" on Mac — Add sudo before the command and enter your Mac password.
❓
Vercel build fails — Go to your Vercel dashboard → click the failed deployment → read the error log → the most common fix is making sure vercel.json is in the root folder.
❓
App looks broken on phone — Make sure you're visiting the https:// version of your Vercel URL, not http.
